# Firebase Function 101

This week Bikram Tuladhar shared Google Assistant cloud implementation in Firebase cloud function as serverless architecture. Firebase function is usually triggered by HTTP event and developers don't have to manage server and free of worry about scaling.

In the session, Bikram Tuladhar demonstrates the web query to Google Assistant using Firebase cloud. Below there is code repository for his demo:

[bikramtuladhar/google-assitance-in-firebase-function](https://github.com/bikramtuladhar/google-assitance-in-firebase-function)

Here is a Firebase cloud function [URL](https://us-central1-chatsansar-ae220.cloudfunctions.net/assistantdemo?query=what%20time%20is%20it%20?) to test Google Assistant web query.
