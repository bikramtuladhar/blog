# Dotfiles

In UNIX file systems, dotfiles are text-based configurations files that stores settings of almost every application, service and tool running on your system. This week Bikram Tuladhar shared mechanism to automate the system configuration of a system using dotfiles.

He demonstrates the use case of dotfiles including some examples from [freekmurze/dotfiles](https://github.com/freekmurze/dotfiles).

For more reading follow the GitHub page: [https://dotfiles.github.io/](https://dotfiles.github.io/)
